# Mulia Grup Private Management System

Sistem manajemen dokumen dan berbagi file privat untuk internal **Mulia Grup**.

## Fitur Utama

- **Manajemen File**: Unggah dan kelola berbagai format dokumen secara aman.
- **Berbagi Terarah**: Bagikan file ke klien tertentu atau grup klien secara spesifik.
- **Grup Klien**: Kelompokkan klien untuk memudahkan distribusi akses dokumen.
- **Log Aktivitas**: Pantau setiap tindakan penting yang terjadi di sistem.
- **Notifikasi**: Pemberitahuan email otomatis saat berkas dibagikan.
- **Antarmuka Modern**: Dashboard yang bersih, responsif, dan mudah digunakan (Desktop & HP).
- **Lokalisasi**: Antarmuka sepenuhnya dalam Bahasa Indonesia.

## Struktur Sistem

- **Portal Admin**: Untuk staf sistem mengelola file, kategori, dan klien.
- **Portal Klien**: Untuk klien melihat dan mengunduh berkas yang dibagikan kepada mereka.

## Teknologi

Sistem ini dibangun menggunakan framework PHP modern untuk memastikan keamanan dan performa yang optimal.
